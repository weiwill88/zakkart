## 三、技术风险备忘（开发过程中关注）

| # | 风险 | 影响阶段 | 应对策略 |
|:-:|------|:-------:|---------|
| R1 | 云函数冷启动慢（1-3s） | 全程 | 单入口函数缓解；上线后加定时保活 |
| R2 | 云函数单次运行 60s 超时 | 阶段 2 | 下单生成需评估耗时，必要时拆分 |

| R4 | NoSQL 聚合查询性能 | 阶段 6-7 | 冗余字段 + 索引 |
| R5 | 阶段 3/4 接口兼容 assembly_order | 阶段 3-4 | 验货/发货接口同时支持 batch_part 和 assembly_order |
| R6 | 云数据库事务限制（10 次/事务） | 阶段 5 | BOM 最多 5-6 种配件，约 7-8 次操作，阶段 0 实测 |
| R7 | Web 安全域名配置 | 阶段 0 | 控制台未找到入口，需实测 JS SDK 在 localhost 的表现 |
